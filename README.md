How to run the project in vs code ?

Step 1: Right click on server folder and then select option "Open in Intergrated Terminal".
Step 2: In terminal run command "npm install" and then what till all the paclages are installed.
Step 3: Run command " npm start "

Step 4: On the top of the screen click on "Terminal" and then select "New Terminal" from the dropdown list.
Step 5: Again run "npm install" Now the packages will install in main project folder, earlier we installed packages in server folder.
Step 6: run command " npm run dev" and now you will get a localhost link. Click on the link to run the project in browser.

Step 7: There are 4 dashboards, To access these dashboards just type " / and the name of the dashboard like kitchen, admin, counter, etc.
